# Workshop Thursdays Poster Template

This template is for creating consistent Thursday workshop posters for live Zoom sessions.

## Template Structure

Use this prompt template structure when generating Thursday workshop posters:

```
Create a marketing poster for a live Zoom workshop session.

CRITICAL: ALL text, titles, descriptions, labels, and content MUST be in Hebrew. The entire poster must be in Hebrew language.

Brand: Aviz - The Architect - Teaching non-tech people to code with AI and architectural understanding.

Colors: Use #22C55E (friendly green) for primary accents and CTAs, with supporting grays #374151, #4B5563, #6B7280, #9CA3AF, #111827. Use secondary greens #16A34A, #158235, #166534, #14532D for depth and variation.

Style: Professional, educational, tech-forward aesthetic with clean typography and modern design.

Layout Requirements:
- Consistent layout structure across all Thursday workshop posters
- The avatar (The Architect) should be featured doing something wild and engaging with a great hook
- Visual should be eye-catching and convey the workshop topic through the avatar's action/context
- Maintain professional yet dynamic feel

Poster Content:
- Title: [WORKSHOP_TITLE] (in Hebrew)
- Description: [WORKSHOP_DESCRIPTION] (in Hebrew)
- Date/Time: יום חמישי - Zoom (Thursday - Zoom) or specific date if known
- Visual: Avatar (The Architect) [SPECIFIC_ACTION/CONTEXT related to workshop topic] - make it wild, engaging, and hook-worthy

Additional Notes:
- The avatar should be incorporated creatively into the design, not just placed as a static element
- Use the avatar doing something related to the workshop topic - make it memorable and attention-grabbing
- Ensure Hebrew text is prominent and readable
- Maintain brand consistency with green accent colors
```

## Usage Instructions

When creating a new Thursday workshop poster:

1. Replace `[WORKSHOP_TITLE]` with the workshop title in Hebrew
2. Replace `[WORKSHOP_DESCRIPTION]` with a brief description of the workshop in Hebrew
3. Replace `[SPECIFIC_ACTION/CONTEXT]` with a creative description of what The Architect should be doing in the image that relates to the workshop topic
4. Update date/time if specific date is known (otherwise use generic "יום חמישי - Zoom")
5. Use the complete prompt to generate the poster

## Example Usage

**Workshop Topic:** Introduction to AI Coding

**Title:** מבוא לקידוד עם AI (Introduction to AI Coding)

**Description:** למד איך להשתמש ב-AI ליצירת קוד עם הבנת ארכיטקטורה (Learn how to use AI to create code with architectural understanding)

**Visual Action:** The Architect coding at a futuristic terminal with AI assistants floating around, looking excited and dynamic

## Brand Consistency

Remember to always:
- Use "Aviz - The Architect" as the brand name
- Include the avatar image (automatically uploaded by the script)
- Use the exact hex color codes provided
- Keep all content in Hebrew
- Maintain the professional, educational, tech-forward style
- Feature The Architect doing something wild and engaging
