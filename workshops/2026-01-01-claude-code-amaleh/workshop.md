## Overview

סדנה פרקטית שחושפת את Claude Code - לא עוד צ'אטבוט, אלא סוכן אמיתי שמסוגל לייצר **10x תפוקה** במגוון תחומים, גם כאלה שרחוקים לגמרי מתיכנות.

### הטענה המרכזית
Claude Code הוא הצצה לסוכן כללי אמיתי - כזה שעד היום דיברו עליו אבל טרם ראו אותו עובד.

### המפתח: "קלודביליות"
הדרישה המרכזית היא להפוך משימות ל"קלודביליות" - ניתנות לביצוע End-to-End על ידי Claude Code. ברגע שאנחנו צריכים להתערב במרחב הפעולה - יש צוואר בקבוק ואין האצלה אמיתית.

### הגישה של Anthropic
Claude Code הוא לא סוכן אוטונומי לחלוטין - הוא העובד במשרד שאנחנו מציצים על העבודה שלו פעם בחצי שעה-שעה. מצד אחד יכול לרוץ באופן אוטונומי, ומצד שני מסתנכרן איתנו כל הזמן.

### מה נבנה בסדנה
נבנה יחד Skill שעובד - מאפס ועד הרצה. נלמד Context Engineering, כתיבת Skills, וחיבור Plugins.

## Key Messages

### מה Anthropic אומרים על Claude Code

> "Claude Code is far more than a coding tool. At Anthropic, we've been using it for deep research, video creation, and note-taking, among countless other non-coding applications."

> "Users report discovering workflows that multiply productivity by 10x or more."

> "The closest thing we have to an everything agent."

### האסטרטגיה של Anthropic

- מאז Claude 3.5 - פוקוס על יכולות אגנטיות
- Computer Use - שליטה במחשב כמו בן אדם
- MCP - פרוטוקול פתוח לתקשורת בין סוכנים
- Skills (דצמבר 2025) - סטנדרט פתוח ליכולות מודולריות

### למה דווקא קוד?

Anthropic לקחו את ה-use case של קוד והסתכלו דרכו על עבודה של סוכן באופן כללי:
- ניהול קבצים = זיכרון לטווח ארוך
- הרצת סקריפטים = הורדת משימות מכאניות
- MCP = חיבור לעולם החיצון
- Skills = יכולות מותאמות אישית
- Hooks = אוטומציות
