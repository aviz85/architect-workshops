## Overview

Learn how to leverage Claude Code CLI to boost productivity and automate workflows in your business.

סדנה מעשית ללמוד איך להשתמש ב-Claude Code CLI כדי להגביר את הפרודוקטיביות ולבצע אוטומציה לתהליכי עבודה בעסק שלכם. נלמד איך להשתמש בכלי המתקדם הזה דרך שורת הפקודה כדי לחסוך זמן, לשפר תהליכים ולהפוך משימות מורכבות לפשוטות.

---

## Target Audience

- בעלי עסקים ומנהלים המעוניינים לשפר תהליכי עבודה
- אנשי מקצוע שרוצים להגביר את הפרודוקטיביות האישית
- מי שיש לו ידע בסיסי בעבודה עם מחשב
- רמה: מתחיל-בינוני (אין צורך ברקע טכני מתקדם)

## Agenda

1. **מהו Claude Code ולמה זה רלוונטי לעסק?** - 20 דקות
   - הבנת הכלי והיכולות שלו
   - מה אפשר לעשות עם Claude Code במציאות העסקית

2. **התקנה והגדרה ראשונית** - 30 דקות
   - התקנת Claude Code CLI
   - חיבור ל-API
   - הגדרות בסיסיות

3. **שימושים עסקיים מעשיים** - 60 דקות
   - אוטומציה של משימות חוזרות
   - יצירת תוכן עסקי (מיילים, דוחות, מסמכים)
   - ניתוח נתונים ויצירת דשבורדים
   - אינטגרציה עם כלי עבודה קיימים
   - תרגול מעשי

4. **Skills ו-Hooks למקרים עסקיים** - 30 דקות
   - בניית skills מותאמים אישית לעסק
   - דוגמאות: טיפול במיילים, ניהול פרויקטים, דיווח אוטומטי

5. **שאלות ותשובות + Next Steps** - 20 דקות
