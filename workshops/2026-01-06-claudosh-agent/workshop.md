## Overview

A beginner-friendly workshop that makes advanced AI agents accessible to everyone. Instead of diving into technical details, we start with 7 jaw-dropping real-world examples that demonstrate what's possible. Then we reveal the surprisingly simple method behind it all.

The key insight: Claude Code isn't just another AI assistant - it's an agent that can work autonomously for extended periods. Understanding this difference changes everything.

## Agenda (90 min)

### Part 1: Mind-Blown - 7 Case Studies (15-20 min)
**Goal:** Hook the audience with jaw-dropping results before explaining the method

7 real examples of crazy things done with Claude Code:

1. **This Workshop System** (Meta!)
   - Skills for poster generation, WhatsApp automation, marketing copy
   - Claude manages the workshops where I teach about Claude
   - Show the folder structure, skills, how it all connects

2. **AI Video Maker** - Full Production Pipeline
   - 8 phases: Story → Storyboard → References → Frames → Upscale → End Frames → Videos → Edit
   - Review system with approval workflow
   - Cost tracking and estimation
   - Multiple API integrations (Gemini, Kling, Grok)

3. **Daf Yomi** - Daily Talmud Content Automation
   - Fetches today's page from Sefaria API
   - Generates Torah insights (vortim) in Hebrew
   - Creates branded posters automatically
   - Religious content with proper RTL handling

4. **WAHA Stack** - WhatsApp Automation Platform
   - Full SaaS: React dashboard, Express backend, SQLite
   - Gemini AI chatbot with session management
   - Voice messages with ElevenLabs TTS/STT
   - Docker deployment with CI/CD pipeline
   - Even includes an arcade game!

5. **Production Analytics System** (Private Project)
   - GA4 + BigQuery integration for mobile app
   - User journey tracking, funnel analysis
   - Screen flow visualization
   - Deep data exploration tools

6. **Frontend Quick Launcher** (gh-pages-deploy skill)
   - One command to publish ANY frontend to the web
   - Creates GitHub repo, enables Pages, returns live URL
   - Demos, prototypes, visualizations - instantly shareable
   - No manual setup, no config files, just "publish this"

7. **GTD Task Manager** - מערכת ניהול משימות בפרומפט אחד
   - ביקשתי "מערכת ניהול משימות" - קלוד הציע מערכת מבוססת GTD
   - בנה מערכת מלאה עם כל הפקודות והסקילים
   - Inbox, Next Actions, Projects, Contexts - הכל עובד
   - דוגמה מושלמת ל"תן לקלוד להציע" במקום לתכנן מראש
   - [הסבר על המערכת](https://aviz85.github.io/claude-tasks/)

Key message: "All of this is possible thanks to ONE technique I'm about to show you"

### Part 2: The Breakthrough (15 min)
**Goal:** Explain what changed - the paradigm shift

- What's the "good news" (בשורה) of this method?
- Comparison to everything before: ChatGPT, Copilot, traditional assistants
- The crucial difference: **Long time horizon of autonomous work**
- From "AI assistant" to "AI agent that works FOR you"

### Part 3: Claudible or Not? - Interactive Session (15 min)
**Goal:** Build intuition for what tasks can be delegated

- Interactive riddle with the audience
- Present various tasks - audience guesses: Claudible or Not-Claudible?
- Develop a mental framework for task delegation
- What makes a task suitable for autonomous AI work?

### Part 4: The Method - Step by Step (30-40 min)
**Goal:** Hands-on walkthrough of the working method

- **Project = Folder** - every project lives in its own folder
- **The `.claude` folder** - the project's brain and memory
- **CLAUDE.md** - project-specific instructions
- **Skills** - building reusable capabilities as you work
- **Live demo** - showing a real project workflow
- Building skills through ongoing work (not upfront)

### Part 5: Q&A and Wrap-up (10 min)
